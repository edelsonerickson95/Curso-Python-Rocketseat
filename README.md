# Curso-Python-Rocketseat
Desafio prático - 1

Sobre o desafio
Neste desafio, você reforçará de forma prática os conceitos que aprendemos no módulo "Introdução ao Python". Como se trata de um desafio, ele necessita de alguns conhecimentos além dos abordados nesse módulo, então é importante ter autonomia para conseguir pesquisar essas coisas caso não saiba como resolver. Por isso, lembre-se, tenha calma e acredite no seu processo.
